# File Share App Advanced

## Setup
1. Install Python 3.10
2. pip install -r requirements.txt
3. python app.py

## Features
- Encrypted uploads (AES-256 GCM)
- Share via secure code or QR
- Supports multiple files/folders (zipped)
- Expiry time (in minutes)
- Max download count
